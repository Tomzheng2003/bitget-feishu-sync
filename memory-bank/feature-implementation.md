## 未来规划 (暂不实施)

### 飞书消息通知
**优先级**: 中

- 开仓提醒: 发送币种、方向、杠杆、入场价
- 平仓提醒: 发送收益额、收益率、持仓时长
- 异常报警: API 连接失败时告警

### 数据可视化看板
**优先级**: 低

- 资金曲线: 累计收益走势图
- 胜率分析: 按周/月统计
- 币种偏好: 交易频次排行

### Binance 支持
**优先级**: 低 (代码已实现, 暂时禁用)

- 多交易所统一管理
- Binance Futures API 对接
